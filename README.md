## Building and running on localhost

First install dependencies:

```sh
npm install
```

Run with parcel
```sh
parcel src/index.html
```

Production Building
```sh
parcel build src/index.html
```

To run in hot module reloading mode:

```sh
npm start
```

To create a production build:

```sh
npm run build-prod
```

## Running

```sh
node dist/bundle.js
```

## Testing

To run unit tests:

```sh
npm test
```
